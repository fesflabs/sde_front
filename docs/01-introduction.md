# Next.js Feature-Based Architecture Boilerplate

Este boilerplate fornece uma base robusta para a construção de aplicações React modernas e escaláveis usando Next.js com uma arquitetura baseada em funcionalidades. Ele incorpora as melhores práticas da indústria e uma stack tecnológica cuidadosamente selecionada para ajudar equipes de desenvolvimento a construir aplicações de alta qualidade de forma eficiente.

## Visão Geral

Este boilerplate foi projetado com foco em:

- **Organização Estrutural**: Através de módulos que refletem os setores da empresa, cada um contendo suas próprias funcionalidades
- **Manutenibilidade**: Através de uma organização clara do código e separação de responsabilidades
- **Controle de Acesso**: Gerenciamento de permissões em nível de módulo e funcionalidade
- **Escalabilidade**: Organizando o código em módulos e funcionalidades independentes que podem crescer sem aumentar a complexidade
- **Performance**: Utilizando estratégias eficientes de gerenciamento de estado e busca de dados
- **Experiência do Desenvolvedor**: Com ferramentas abrangentes e documentação
- **Testes**: Incluindo uma estrutura de testes bem estruturada e abordagem prática

## Tecnologias Principais

| Categoria                      | Tecnologia          | Descrição                                                                               |
| ------------------------------ | ------------------- | --------------------------------------------------------------------------------------- |
| **Framework**                  | Next.js 15.2.3      | Framework React com App Router                                                          |
| **Biblioteca UI**              | Shacnui +Tailwind   | Biblioteca UI React moderna, personalizável e com os recursos mais recentes do Tailwind |
| **Gerenciamento de Estado**    | Zustand 5           | Solução leve para gerenciamento de estado                                               |
| **Busca de Dados**             | TanStack Query      | Poderoso sistema de busca e cache de dados                                              |
| **Manipulação de Formulários** | React Hook Form     | Gerenciamento eficiente de formulários                                                  |
| **Validação**                  | Zod                 | Validação de esquemas com prioridade para TypeScript                                    |
| **Cliente HTTP**               | Axios (via adapter) | Cliente HTTP configurável com padrão de adapter                                         |
| **Testes**                     | Vitest              | Framework de testes moderno                                                             |
| **Linting**                    | ESLint              | Qualidade de código e consistência                                                      |
| **Formatação**                 | Prettier            | Formatação de código                                                                    |
| **Git Hooks**                  | Husky               | Hooks git automatizados                                                                 |
| **Autenticação**               | Custom JWT          | Autenticação baseada em token com middleware                                            |

Esta documentação fornece informações abrangentes sobre a arquitetura, funcionalidades e fluxo de desenvolvimento para construir aplicações com este boilerplate.
